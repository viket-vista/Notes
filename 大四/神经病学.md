# 总论
## 症状
### 头痛
+ 问诊方式
	+ 部位
	+ 形式
	+ 性质
	+ 加重因素
	+ 程度
	+ 伴随
	+ 先兆
+ 产生部位
	+ 颅外病变：与病灶一致
	+ 颅内病变：与病变部位不精确
		+ 小脑幕以上的病变同侧痛
		+ 以下肿瘤位于后枕部痛
		+ 感染呈全头痛
### 眩晕
+ 眩晕：自身或周围物体旋转、漂浮或翻滚
	+ 性质
		+ 真性眩晕：自身对外界空间位置错觉
		+ 假性眩晕：仅有晕动感
	+ 部位
		+ 系统性眩晕：周围性眩晕、中枢性眩晕，前庭神经系统
		+ 非系统性眩晕：眼部、心血管、内分泌疾病、中毒感染贫血 
+ 头晕：头重脚轻，眼花和站立不稳
+ 头昏：脑子昏昏沉沉
### 感觉异常
+ 抑制性症状
	+ 完全性感觉障碍：一个部位各种感觉缺失
	+ 分离性感觉障碍：意识清醒、某部位某种感觉障碍而其他保存
	+ 皮质感觉缺失：深浅感觉正常，无视觉不能辨别形状重量
+ 刺激性或
